# Nolseoul 🎡  
**서울에서 놀다, Nolseoul**  
디자인부터 브랜딩, UI 설계 및 프론트엔드 구현까지  
전 과정을 직접 기획한 개인 프로젝트입니다.

---

## 📌 프로젝트 소개 

**놀서울(NolSeoul)**은 서울시의 문화행사, 문화공간, 야경명소 정보를 통합 제공하는 웹 애플리케이션입니다.  
서울시 공공 API를 활용해 실시간 데이터를 제공하며, 위치 기반 지도 보기, 행사 상세 모달 등 사용자 중심의 기능을 구현했습니다.

- **로고 & 네이밍 직접 제작**  
- **디자인 감각을 살린 UX/UI 구성**  
- React 기반의 컴포넌트 단위 설계  
- **반응형 웹** 구성으로 다양한 디바이스 대응

---

## 🔧 주요 기능

- 🧭 문화행사, 공간, 야경 정보 페이지 구성  
- 🔍 카테고리 필터 및 리스트 탐색, 상세 모달 구현  
- 🗺 KakaoMap 기반 지도 위치 표시 기능  
- 🎨 직접 설계한 사용자 중심 UI 디자인  
- 📱 모바일 환경 대응을 위한 반응형 스타일링

## 🔍 문제 해결 경험 요약

- 🗺 **지도 위치 오류**  
  → 주소 기반 지도가 표시되지 않아 좌표 기반으로 전환, 위경도 필드 오류 수정

- 🗂 **카테고리 필터 문제**  
  → 누락된 CODENAME 대신 텍스트 키워드 기반 필터링 직접 구현

- 🚫 **Vercel 배포 시 지도/API 오류**  
  → 환경변수 분리 및 클라이언트 사이드 조건 처리로 해결

- 📦 **공공데이터 누락 대응**  
  → 조건부 렌더링(`&&`)을 통해 빈 값에도 UI가 깨지지 않도록 방어적 설계

- 🌀 **슬라이더 반응형 오류**  
  → Swiper breakpoints 옵션과 Tailwind 반응형 유틸리티로 해결
